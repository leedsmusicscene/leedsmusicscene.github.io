Slaves, Wild Beasts and White Lies announced as first acts for Live at Leeds 2017

This article was published in 2016.

The award winning metropolitan festival, Live at Leeds, returns on Saturday 29th April 2017 and the first bands have been announced this week, including Slaves, Wild Beasts, White Lies, Jagwar Ma and The Pigeon Detectives.

The full list of bands announced this week for Live at Leeds 2017 are:

SLAVES // WILD BEASTS // WHITE LIES // THE PIGEON DETECTIVES // JAGWAR MA // TELEMAN // SUPERFOOD // BLACK HONEY // THE MAGIC GANG // THE AMAZONS // THE BIG MOON // JAWS // LET'S EAT GRANDMA // BAD SOUNDS // BLACKWATERS // BRIAN DEADY // CLAY // COLOURING // FANGCLUB // FLAMINGODS // FREAK // GURR // KLLO // LAMIA // LUXURY DEATH // MARSICANS // NGOD // NILUFER YANYA // TEAM PICTURE // THE BAY RAYS // THE NIGHT CAFÉ // THE OPERA COMIC // THE PALE WHITE

Andy Smith of Live at Leeds said, "It's great to welcome back local heroes the Pigeon Detectives and Wild Beasts, it's been 7 years since Wild Beasts last joined us. Alongside them is a plethora of exciting established and new breaking artists."

Many more bands are yet to be announced as the festival will once again take over a variety of venues and spaces throughout the city and aim to showcase the best new emerging artists alongside more established acts from around the world.

Live At Leeds 2017 tickets are on sale now via Leeds Gig Tickets priced at £32.50.
