Complete line up for NME / Radio One Stage at Leeds Festival 2012 revealed

This article was published in 2012.

The full line up for the NME / Radio One Stage has been announced.

Leeds 2012 takes place at Bramham Park over the August Bank Holiday. Headliners on the Main Stage this year are Foo Fighters (Friday), The Cure (Saturday) and Kasabian (Sunday). The Lock Up Stage will be headlined by Less Than Jake (Friday) and Social Distortion (Saturday) with the Dance Stage headlined by Metronomy (Sunday).

The NME / Radio One Stage line up is as follows:

FRIDAY: Justice // Two Door Cinema Club // The Horrors // SBTRKT // Mark Lanegan Band // The Joy Formidable // Tribes // Django Django // Howler // Of Monsters And Men

SATURDAY: The Maccabees // Foster The People // The Courteeners // Graham Coxon // Passion Pit // The Blackout // The Hives // Spector // Friends // Chiddy Bang // Here We Go Magic

SUNDAY: At The Drive-In // The Cribs // Mastodon // Billy Talent // Miike Snow // Santigold // Dry The River // Twin Atlantic // Pure Love

The winner of the annual Futuresound competition will open the stage at Leeds on the Sunday lunchtime.

More bands are still to be announced including the full line up for the Festival Republic Stage and the BBC Introducing Stage.
