Many more bands added to Leeds Festival 2003 line up...

This article was published in 2003.

More Main Stage and Radio One Stage bands have been announced today for Leeds Festival 2003. Those bands include: Staind, The Darkness, Cave In, The Thrills, The Kills, Jet, Grandaddy, Hell is for Heroes, The Eighties Matchbox B-Line Disaster, Radio 4 and The Raveonettes.

In addition, the line up for the Concrete Jungle Stage has been announced, and is to be headlined by Pennywise.

Full details More acts are to be announced over the coming weeks.

Tickets cost £95 for the weekend, or £40 for one day. For more information or to purchase tickets, visit Ticketmaster.

Those line ups in full:-

FRIDAY MAIN STAGE: Metallica / System of a Down / Sum 41 / Primal Scream / Courtney Love / Good Charlotte / Lost Prophets / The Used
FRIDAY RADIO ONE STAGE: The Music / Yeah Yeah Yeahs / Hot Hot Heat / Hell is for Heroes / The Eighties Matchbox B-Line Disaster / Radio 4 / The Raveonettes

SATURDAY MAIN STAGE: Linkin Park / Blink 182 / Placebo / Jay Z / Staind / The Datsuns / Finch / Bowling For Soup
SATURDAY RADIO ONE STAGE: The Polyphonic Spree / Elbow / Interpol / Electric Six

SUNDAY MAIN STAGE: Blur / White Stripes / Beck / Doves / The Streets / The Libertines / Junior Senior / The Darkness / Cave In
SUNDAY RADIO ONE STAGE: AFI / Hundred Reasons / Cooper Temple Clause / The Mars Volta / The Thrills / The Kills / Jet
SUNDAY CONCRETE JUNGLE STAGE: Pennywise / Lagwagon / The Ataris / Alkaline Trio / Mad Caddies / Turbonegro / Thrice / King Prawn / Funeral for a Friend

Check www.leedsfestival.com for more details.
