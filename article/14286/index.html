More bands announced for Live At Leeds 2011

This article was published in 2011.

More bands have been announced this morning for the Live At Leeds 2011 festival.

The official launch of Live At Leeds 2011 festival took place last Friday, with over 60 bands announced for the festival already.

On Sunday 1st May, Glasvegas will headline at the Leeds Metropolitan University as part of the Live At Leeds festivities. You will need to buy a separate ticket for this event - buy from See Tickets.

The other bands announced today - to appear on Saturday 30th April - are:

JAMES BLAKE // FRIGHTENED RABBIT // THE FUTUREHEADS // JON FRATELLI // FIGHT LIKE APES // CATHERINE AD // SAINT SAVOIR // STAR SLINGER // LET'S BUY HAPPINESS // PAPER CROWS // THE KILL VAN KULLS // MARMOZETS // POST WAR GLAMOUR GIRLS // THE ABC CLUB // BEAR MASK
