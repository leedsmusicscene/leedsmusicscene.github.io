More bands added to Slam Dunk Festival 2015 line up

This article was published in 2015.

Slam Dunk Festival will be taking over Leeds City Centre this coming May Bank Holiday with a spectacular event that spans stages around Millennium Square, O2 Academy and Leeds Beckett University.

This week organisers confirmed the addition of more bands, including Don Broco, Pvris, While She Sleeps and The Bronx, who join a line up that already includes headliners You Me At Six, Lower Than Atlantis and Taking back Sunday.

The line up so far includes:

A LOSS FOR WORDS // BANE // BAYSIDE // BEARTOOTH // BIG D & THE KIDS TABLE // COMEBACK KID // CROSSFAITH // DON BROCO // EMMURE // FIREWORKS // GALLOWS // GOLDFINGER // H2O // KNUCKLE PUCK // LIGHTYEAR // LOWER THAN ATLANTIS // MARIACHI EL BRONX // MILLENCOLIN // NECK DEEP // PATENT PENDING // PVRIS // REEL BIG FISH // SUCH GOLD // TAKING BACK SUNDAY // THE BRONX // THE COLOR MORALE // THE WONDER YEARS // THY ART IS MURDER // TRANSIT // TRASH TALK // WHILE SHE SLEEPS // YOU ME AT SIX // ZEBRAHEAD

Slam Dunk Festival 2015 takes place on Saturday 23rd May at venues in Leeds City Centre (Millennium Square, O2 Academy, Leeds Beckett University, Ba Bar and 2 other outdoor stages).
