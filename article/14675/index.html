New Additions to Beacons Festival 2011

This article was published in 2011.

Frankie and the Heartstrings have been added to this year's Beacons Festival and will be opening the event a day early. The group will be opening the festival on Thursday, August 11th exclusively to weekend and Friday ticket holders, stretching the festival across 4 days.

The full line up so far is Jamie Woon / Mr Scruff / Jamie XX / Frankie and the Heartstrings / The Phantom Band / Toddla T / Hudson Mohawke / Summer Camp / Mount Kimbie / The Twilight Sad / Willy Mason / Emmy The Great / Factory Floor / The Cave Singers / Dutch Uncles / Dry The River / Admiral Fallow / Man Like Me / Ducktails / Pearson Sound / Optimo / The Apples / Anika / Polar Bear + MC Jyager / Pariah / Ghostpoet / Andy Votel / Mazes / Star Slinger / Dog Is Dead / Spectrals / Islet / D/R/U/G/S / Benjamin Francis Leftwich / Girl Unit / Fantastic Mr Fox / Wolf People / I Like Trains / Demdike Stare / My Tiger My Timing / Swimming / Eagulls / Paper Crows / Hot Horizons / Napoleon IIIrd / Ellen and the Escapades / Runners / Juffage / Nope, Lone / That Fucking Tank / The Horn The Hunt / Wonderswan / Milk White White Teeth / Blood Oranges / Pengillys / Worriedaboutsatan / Serious Sam Barrett / Sam Sallon and many more.
